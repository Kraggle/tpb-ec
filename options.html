This will give the preferred if available, if not it will give 720p if 1080p, 1080p if 720p is not available and then Standard otherwise. Also be aware that if ONLY a higher quality is available and you have selected a lower quality, it will give you the better link.

Allow HEVC (x265)?

If this is true, it will be preferred over non-hevc but will still defer to non-hevc if none exist. Also if this is set to false, it will never give you hevc, even if it is the only one that exists.

Auto set as watched?

This option will, if the link is available, set the episode as watched!
